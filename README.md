[VSCode](https://code.visualstudio.com/).

## Project Setup

```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn start
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```

### Generate Types

```sh
yarn typgen
```

### Generate snaplet seeder

```sh
yarn snaplet
```
